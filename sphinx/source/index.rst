Documentation for dsa110-calib
================================

dsacalib is a calibration pipeline for the `DSA-110 array <https://www.deepsynoptic.org/home>`_ at the Caltech's Owen's Valley Radio Observatory.  dsacalib is built on CASA for automated and offline calibration of radio interferometric data.
